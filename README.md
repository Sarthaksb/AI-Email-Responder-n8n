# 🤖 AI Email Responder using n8n + Google Gemini

An AI-powered email automation workflow built with n8n and Google Gemini.

## 🚀 Features

- Monitors new unread Gmail messages
- Extracts sender, subject, and email content
- Uses Google Gemini AI to analyze incoming emails
- Classifies emails based on category and urgency
- Generates professional replies automatically
- Sends the reply through Gmail
- Marks processed emails as read

## 🛠️ Tech Stack

- n8n
- Google Gemini API
- Gmail API
- AI Automation

## 📂 Workflow

```
New Email
      ↓
Extract Email Data
      ↓
Google Gemini AI
      ↓
Generate Smart Reply
      ↓
Send Gmail Reply
      ↓
Mark Email as Read
```
